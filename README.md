# EventSpark Frontend

Frontend application for **EventSpark**, Pakistan's #1 local events discovery platform. Built with **React**, it allows users to browse, search, and register for events, as well as contact event organizers.

---

## 🚀 Live Demo

[Frontend Live](https://event-spark-frontend.vercel.app/)

---

## 📁 Project Structure

EventSpark-Frontend/
├── public/                # Public assets (images, favicon, etc.)
├── src/
│   ├── api/               # Axios instance and API calls
│   ├── components/        # Reusable components (Navbar, Footer, EventCard, etc.)
│   ├── hooks/             # Custom React hooks (useEvents, etc.)
│   ├── pages/             # Page components (Home, Events, Contact, AttendanceForm)
│   ├── routes/            # React Router configuration or route-related components
│   │   └── AppRoutes.jsx  # Example: central routing component
│   ├── styles/            # CSS Modules
│   └── App.jsx            # Main App component (renders AppRoutes and layout)
├── package.json
├── package-lock.json
└── README.md

---

## ⚙️ Technologies Used

- **React** – Frontend UI library  
- **React Router DOM** – Routing for SPA  
- **Axios** – HTTP requests to backend API  
- **CSS Modules** – Component-level styling  
- **Lucide Icons** – For vector icons in UI

---

## 📦 Features

- Browse upcoming events with filters by category & search  
- Detailed event pages with registration links  
- Contact form to reach event organizers  
- Attendance registration form  
- Responsive design for desktop & mobile  
- Testimonials and social media links
